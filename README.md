Strong password Generator using Python

This Python program creates a secure, random password of a specified length (default is 15 characters). It uses the random and string modules to combine uppercase and lowercase letters, digits, and punctuation symbols into a pool of possible characters. The generate_password function randomly selects characters from this pool and joins them into a single string. Finally, the generated password is printed to the console.
